Hazard Viewer

Interactive global earthquake probability maps

Access real-time earthquake forecasts for any location worldwide. Our proprietary Pattern Informatics algorithm provides probability assessments for earthquakes of magnitude 5 and above, updated daily with the latest seismic data.

Key Features

  • Global coverage with 0.1° grid resolution
  • Customizable time periods (3 months to 3 years)
  • Magnitude threshold selection (M5 to M8+)
  • Daily updates at 21:30 PT
  • Ground shaking intensity overlays
  • Historical earthquake data visualization
